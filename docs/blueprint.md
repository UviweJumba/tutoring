# **App Name**: ModuleMastery

## Core Features:

- Landing Page with Module Previews: Showcase a clean, informative landing page with previews of available modules, enticing potential students.
- Syllabus Hub Navigation: Implement a Syllabus Hub page with a grid layout for easy navigation and discovery of all available modules.
- Individual Module Pages: Create dedicated module pages with free preview content and a clear pathway to unlock premium content.
- Contact Integration: Incorporate mailto: links strategically throughout the app for easy communication regarding premium access.

## Style Guidelines:

- Primary color: Muted teal (#66A5AD) for interactive elements and highlights.
- Background color: White (#FFFFFF) for a clean and calm learning environment.
- Text color: Dark gray (#333333) for primary text, softer gray (#777777) for secondary text.
- Font: 'Inter' (sans-serif) for a clean and highly readable experience.
- Ample whitespace and padding to provide a calm and structured feel. Two-column layout on desktop, stacked on mobile for module pages.
- Use simple, professional icons for module categories.
- Subtle transitions when navigating between modules.
